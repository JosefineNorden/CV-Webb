Padelbokningssystem

Beskrivning av Padelbokningssystemet:

PadelBookingAPI är ett REST-API för bokning av padelbanor med tre banor. Systemet är byggt i .NET 8 med:

  • Entity Framework Core
  • Repository Pattern
  • Swagger
  • Postman
  • xUnit (enhetstester)

Systemet innehåller följande funktioner:

  • CRUD för kunder och bokningar
  • Validering av bokningar (tider, dubbelbokning, öppettider)
  • Se bokningar för en viss dag
  • Visa tillgängliga tider för ett datum
  • Räkna antal bokningar mellan två datum

Testade user stories:

  • Som kund kan jag se lediga tider
  • Som kund kan jag boka en bana i en timme
  • Som admin kan jag se bokningar en viss dag
  • Som admin kan jag se antal bokningar mellan datum

Systemet tillåter:

  • Endast hela timmar
  • Endast bokning mellan 07:00–22:00
  • Ej dubbelbokning

Testning:

  • Enhetstester i xUnit
  • API-testning i Postman
